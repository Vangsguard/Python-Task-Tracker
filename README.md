# Python Task Manager
A terminal based task manager made in python.

### Dependencies
```
pip install rich typer 
```

#### Create Task
```
python todocli.py add "Task" "Category"
```
#### Remove Task
``` 
python todocli.py remove (index)
```
#### Show Task
``` 
python todocli.py show
```
#### Complete Task
```
python todocli.py complete (index)
```
